Integrated Greenhouse Monitor : 
By Bence and Kamiar

Description :
A Website built to display data of temperatures, humidity and soil values recived by muliple 
Soil moisture Sensors to ensure that the areas around them are good for Greenhouses.

How it works :
- Ensure nodejs and npm are installed
- Go to IntegratedGreenhouse directory
- enter command `npm install`
- enter command `node server.js`
- then on browser, go to localhost:2000/greenhouses/Chris'%20Testing
